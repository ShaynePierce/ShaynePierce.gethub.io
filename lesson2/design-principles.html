White Space and Clean Design

Tesla

Tesla uses the open sky in this photo to create an uncrowded scene and simply beautiful user experience. The buttons are few, simple, and makes you want to click for more. Remove the clutter and open up the space in between!

Repetition

T-Mobile

T-Mobile helps its users by repeating the same pattern for each action tile. Start with a related picture, add some explanation text, and drive the user to action. These tiles are repaeated through out the enitre site.

Alignment

Walmart

Walmart helps its shoppers understand what to do by using alignment. Each shopping activity is started with a title aligned-left, then centered icon images and buttons.
